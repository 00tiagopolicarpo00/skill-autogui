# skill-autogui
A Mycroft AI skill to enable keyboard and mouse manipulation via voice command using Autogui

## Usage:
* `type Mycroft is awesome`
* `press enter`
* `scroll up 10 clicks`
* `scroll down 10 clicks`
* `scroll right 10 clicks - Linux only`
